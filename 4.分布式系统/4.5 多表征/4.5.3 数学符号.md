## 1. 分布式系统相关数学符号

- DS：分布式系统
- N：节点集合
- C：通信机制
- R：复制机制
- S：分片机制
- CAP：一致性、可用性、分区容忍性属性
- ⇒：蕴含
- ∀：对所有
- ∃：存在

## 2. 结构化表达

- DS = (N, C, R, S, CAP)
- N = {n1, n2, ..., nn}
- C = {c1, c2, ..., cm}
- R = {r1, r2, ..., rk}
- S = {s1, s2, ..., sl}
- CAP = (Consistency, Availability, PartitionTolerance)
