# 1.3.3 结构图与表

## 1. 系统结构图

- 展示系统各要素、子系统及其关系。
- 常用图类型：层次结构图、网络拓扑图、状态转移图。
- 示例：

```mermaid
graph TD;
  S[系统] --> S1[子系统1];
  S --> S2[子系统2];
  S1 --> E1[要素1];
  S2 --> E2[要素2];
```

## 2. 层次关系图

- 展现系统的多层级结构，反映递归分解。
- 示例：

```mermaid
graph TD;
  Root[系统] --> L1[第一层];
  L1 --> L2[第二层];
  L2 --> L3[第三层];
```

## 3. 关联矩阵与表格

- 用矩阵或表格表达要素间的关系、属性。
- 关系矩阵示例：
|   | E1 | E2 | E3 |
|---|----|----|----|
| E1| 0  | 1  | 0  |
| E2| 1  | 0  | 1  |
| E3| 0  | 1  | 0  |
