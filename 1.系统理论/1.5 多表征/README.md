# 1.5 多表征 (Multiple Representations)

## 概述

多表征是系统理论的重要特征，通过不同的表达方式来描述和理解系统，包括概念图、结构表、数学符号等多种表征形式。多表征不仅提供了理解系统的不同视角，也增强了理论的可理解性和应用性。
