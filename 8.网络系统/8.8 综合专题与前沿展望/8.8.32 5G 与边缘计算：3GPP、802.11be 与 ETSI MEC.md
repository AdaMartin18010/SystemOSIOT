## 1. 5G 系统架构

3GPP 在 TS 23.501 中定义 5G 系统（5GS）架构，采用服务化架构（SBA）：

```text
UE <-> (R)AN <-> 5G Core (5GC)
                ├── AMF（Access and Mobility Management Function）
                ├── SMF（Session Management Function）
                ├── UPF（User Plane Function）
                ├── PCF（Policy Control Function）
                ├── UDM（Unified Data Management）
                └── NRF / NSSF / AUSF / NEF
```

来源：[3GPP TS 23.501](https://www.3gpp.org/specifications)。

## 2. 5G NR 协议栈

5G NR（New Radio）协议栈在 TS 38.300 系列中定义：

| 层 | 功能 |
|---|---|
| SDAP | QoS 流到 DRB 映射 |
| PDCP | 加密、完整性、重排序、重复丢弃 |
| RLC | 分段、重传、级联（AM/UM/TM） |
| MAC | 调度、HARQ、复用/解复用 |
| PHY | OFDM、MIMO、波束管理、 numerology |

来源：[3GPP TS 38.300](https://www.3gpp.org/specifications)。

## 3. 网络切片与边缘计算

- **网络切片（Network Slicing）**：在同一物理基础设施上隔离出多个逻辑网络，满足不同业务（eMBB、URLLC、mMTC）的 QoS 需求。
- **MEC（Multi-access Edge Computing）**：将计算、存储、网络能力下沉到网络边缘，降低延迟并节省回传带宽。

5G 边缘计算通过 UPF 下沉、本地数据网络（Local DN）和 NEF 暴露能力实现。

## 4. IEEE 802.11be / Wi-Fi 7

Wi-Fi 7 在 [IEEE 802.11be-2024](https://standards.ieee.org/standard/802.11be-2024.html) 中标准化，关键特性：

| 特性 | 说明 |
|---|---|
| 320 MHz 信道 | 相比 802.11ax 翻倍带宽 |
| 4096-QAM | 更高频谱效率 |
| Multi-Link Operation（MLO） | 多频段并发传输与负载均衡 |
| Multi-RU | 更灵活的频谱资源分配 |
| Preamble Puncturing | 避免非连续信道干扰 |

## 5. ETSI MEC 框架

ETSI MEC 定义边缘计算参考架构，包括：

- **MEC Host**：承载 MEC 应用与平台的边缘节点。
- **MEC Platform**：提供服务注册、DNS、流量规则、能力暴露。
- **MEC Orchestrator**：跨 MEC Host 编排应用生命周期。
- **CFS Portal**：客户服务门户。

来源：[ETSI MEC Standards](https://www.etsi.org/technologies/multi-access-edge-computing)。

## 6. 5G / Wi-Fi / MEC 协同

```text
终端 ─┬─ 5G NR ─┬─ 5G Core ─┬─ 互联网 / 云
      │         └── UPF(边缘) ─ MEC Host
      └── Wi-Fi 7 AP ─┬─ 本地交换机 ─ MEC Host
                        └── 互联网
```

- 5G 与 Wi-Fi 7 通过 MEC 实现无缝卸载与业务连续性。
- 工业互联网、AR/VR、车联网、云游戏是典型场景。

## 7. 工程部署与安全

- **部署模式**：运营商 MEC、企业专网、公有云边缘节点（AWS Wavelength、Azure Edge Zones、Google Distributed Cloud Edge）。
- **安全挑战**：边缘节点物理安全、切片隔离、UPF 与 MEC 平台接口安全、终端身份认证。
- **权威指南**：NIST SP 800-207（Zero Trust）适用于边缘接入控制；3GPP SA3 定义 5G 安全架构。

## 8. 批判性分析

- **标准碎片化**：3GPP、ETSI MEC、O-RAN、IEEE、ITU-T 各有侧重，跨组织互操作仍复杂。
- **商业模式不成熟**：边缘计算部署成本高，杀手级应用仍在探索。
- **能耗与可持续性**：5G 与 Wi-Fi 7 的高带宽、低延迟以更高能耗为代价，需绿色网络设计。
