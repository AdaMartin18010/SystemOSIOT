# 3.2.2 批判视角

## 1. 批判视角

- **技术本位批判**：过度追求智能化、联网化，忽视用户实际需求与体验。
- **平台垄断批判**：大型平台对生态的控制与数据垄断。
- **安全机制批判**：安全措施复杂化带来的部署与维护难题。
- **能耗与可持续性批判**：低功耗设计与实际能耗矛盾。
- **生态碎片化批判**：标准不统一导致的生态割裂。
