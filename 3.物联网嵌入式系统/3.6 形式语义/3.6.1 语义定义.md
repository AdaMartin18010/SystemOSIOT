# 3.6.1 语义定义

## 1. 语义定义

- 物联网嵌入式系统的语义是指其各组成部分（如节点、传感器、协议等）在形式化模型下的数学意义和行为约束。
- 形式语义强调可验证性、可推理性和一致性。

## 2. 符号化表达

- IoT-ES = (N, S, A, G, C, P)
- 节点状态：Init, Ready, Run, Sleep, Error, End
- 协议集合：P = {p1, p2, ..., pq}

## 3. 结构化表达

- **节点状态机图**：

```mermaid

  Init[初始化] --> Ready[就绪];
  Ready --> Run[运行];
  Run --> Sleep[休眠];
  Sleep --> Ready;
  Run --> Error[故障];
  Run --> End[终止];
```
