>
> **目标**：对比同一 OS 概念在 Linux、Linux PREEMPT_RT、FreeRTOS、Zephyr、QNX 中的实现差异与运行时语义。

---

## 1. 进程/任务

| 概念 | Linux | Linux PREEMPT_RT | FreeRTOS | Zephyr | QNX |
|------|-------|------------------|----------|--------|-----|
| 执行实体 | `task_struct`（进程/线程统一） | 同 Linux | `Task` / `TCB` | `k_thread` / `k_tid_t` | `struct _thread_info` |
| 创建 | `clone()` / `fork()` | 同 Linux | `xTaskCreate()` | `k_thread_create()` | `ThreadCreate()` |
| 销毁 | `do_exit()` / `release_task()` | 同 Linux | `vTaskDelete()` | `k_thread_abort()` | `ThreadDestroy()` |
| 状态 | RUNNING/INTERRUPTIBLE/... | 同 Linux | Ready/Running/Blocked/Suspended | Ready/Running/Blocked/Suspended/DUMMY | READY/RUNNING/BLOCKED/DEAD |
| 上下文 | `mm_struct` + `files_struct` + `nsproxy` | 同 Linux | 仅寄存器 + 栈 | 寄存器 + 栈 + 可选 FPU | 完整寄存器 + 消息队列 |

## 2. CPU 调度

| 概念 | Linux | Linux PREEMPT_RT | FreeRTOS | Zephyr | QNX |
|------|-------|------------------|----------|--------|-----|
| 默认策略 | CFS | CFS + RT（可抢占） | 固定优先级抢占 | 协作/抢占/时间片可选 | 抢占式优先级 + 自适应分区 |
| 实时策略 | SCHED_FIFO / SCHED_RR / SCHED_DEADLINE | 同 Linux（所有代码可抢占） | 固定优先级（隐式 FIFO） | 固定优先级 / 时间片 / 最早截止时间 | 固定优先级 + 分区预算 |
| 优先级范围 | 1~99 RT, 0 CFS nice | 同 Linux | 0~(configMAX_PRIORITIES-1) | 0~CONFIG_NUM_PREEMPT_PRIORITIES | 0~255 |
| 调度延迟 | 数十 us~ms | < 100us（典型） | < 10us | < 10us | < 10us |
| 可调度性分析 | 无内置 | 依赖外部分析 | RMA/EDF 需手动 | RMA/EDF 需手动 | 自适应分区提供预算保证 |
| 调度器入口 | `schedule()` | `schedule()`（中断/系统调用返回均可抢占） | `vTaskSwitchContext()` | `z_swap()` | `SchedNext()` |

---

## 3. 内存管理

| 概念 | Linux | Linux PREEMPT_RT | FreeRTOS | Zephyr | QNX |
|------|-------|------------------|----------|--------|-----|
| 地址空间 | 完整虚拟内存 + MMU | 同 Linux | 通常 FLAT，可选 MPU | 可选 MPU/MMU | 完整虚拟内存 + MMU |
| 页表 | 4/5 级页表 | 同 Linux | 无 | 可选 | 同 Linux |
| 分配器 | Buddy + SLUB/SLAB/SLOB | 同 Linux | heap_1~5 | k_heap / k_mem_slab / k_mem_pool | 标准 C 堆 + 共享内存 |
| 缺页异常 | `handle_mm_fault()` | 同 Linux | 无 | 无（静态映射） | 由进程管理器处理 |
| 内存保护 | MMU + KASLR + PTI | 同 Linux | MPU（FreeRTOS-MPU） | MPU / MMU | MMU + 分区 |

---

## 4. 中断与同步

| 概念 | Linux | Linux PREEMPT_RT | FreeRTOS | Zephyr | QNX |
|------|-------|------------------|----------|--------|-----|
| 中断处理 | top-half + bottom-half (softirq/tasklet/workqueue) + threaded IRQ | 几乎所有 ISR 线程化 | ISR + deferred task | ISR + bottom-half | 用户态 ISR / 事件驱动 |
| 中断延迟 | 数 us~数十 us | < 10us | < 1us~数 us | < 1us~数 us | < 1us~数 us |
| 自旋锁 | `spinlock_t` | `rt_mutex` 替代大部分 spinlock | 关中断 | `k_spinlock`（原子或关中断） | 互斥锁 + 消息传递 |
| 互斥锁 | futex / `mutex_lock()` | `rt_mutex`（支持 PI） | `xSemaphoreTakeMutex()` | `k_mutex_lock()` | `pthread_mutex_lock()` |
| 信号量 | `sem_wait()` / `sem_post()` | 同 Linux | `xSemaphoreTake()` / `xSemaphoreGive()` | `k_sem_take()` / `k_sem_give()` | `sem_wait()` / `sem_post()` |
| 优先级倒置 | futex PI 可选 | rt_mutex 自动 PI | 优先级继承（互斥量） | 优先级继承 | 消息传递天然避免 |

## 5. 文件系统与 I/O

| 概念 | Linux | Linux PREEMPT_RT | FreeRTOS | Zephyr | QNX |
|------|-------|------------------|----------|--------|-----|
| 文件系统 | VFS + ext4/XFS/Btrfs/overlayfs | 同 Linux | FATFS / LittleFS（可选） | FATFS / LittleFS / NVS | QNX 资源管理器 + fs-qnx6 |
| 块 I/O | `submit_bio()` / multi-queue | 同 Linux | 直接驱动 | 直接驱动 | 资源管理器处理 |
| 异步 I/O | `io_uring` / POSIX aio | 同 Linux | 无 | 无 | POSIX aio |
| 设备抽象 | `struct device` / `file_operations` | 同 Linux | 直接寄存器 / HAL | 设备模型 + Devicetree | 资源管理器 |

---

## 6. 网络

| 概念 | Linux | Linux PREEMPT_RT | FreeRTOS | Zephyr | QNX |
|------|-------|------------------|----------|--------|-----|
| 协议栈 | 完整 TCP/IP + netfilter + eBPF/XDP | 同 Linux | FreeRTOS-Plus-TCP / lwIP（可选） | Zephyr 原生网络栈 / lwIP | io-pkt / io-sock |
| Socket API | 完整 BSD socket | 同 Linux | 简化 socket | BSD socket | POSIX socket |
| 网络设备 | `struct net_device` | 同 Linux | 直接驱动 | `struct net_if` | io-pkt 资源管理器 |
| 高并发 | epoll / io_uring | 同 Linux | 无 | poll / select | select / poll |

---

## 7. 安全与隔离

| 概念 | Linux | Linux PREEMPT_RT | FreeRTOS | Zephyr | QNX |
|------|-------|------------------|----------|--------|-----|
| 用户/内核隔离 |  ring 0/3 + syscall | 同 Linux | 可选 privileged/user（ARM） | 可选 userspace | 微内核，驱动用户态 |
| 进程隔离 | MMU + namespaces | 同 Linux | 无 / MPU 区域 | MPU / MMU | MMU + 分区 |
| 资源限制 | cgroups v2 | 同 Linux | 静态配置 | 内存池/堆大小 | 自适应分区 |
| 强制访问控制 | SELinux / AppArmor / Smack / TOMOYO | 同 Linux | 无 | 无 | 无 |
| 安全启动 | UEFI Secure Boot + TPM + IMA | 同 Linux | 取决于平台 | 取决于平台 | QNX Trusted Boot |

## 8. 关键指标对比

| 指标 | Linux | Linux PREEMPT_RT | FreeRTOS | Zephyr | QNX |
|------|-------|------------------|----------|--------|-----|
| 最小内存 | ~16 MB+ | ~16 MB+ | ~4 KB+ | ~8 KB+ | ~1 MB+ |
| 上下文切换 | ~1 us~数 us | ~1 us~数 us | < 1us | < 1us | < 1us |
| 中断延迟 | 数 us~数十 us | < 10us | < 1us | < 1us | < 1us |
| 调度延迟 | 数十 us~ms | < 100us | < 10us | < 10us | < 10us |
| 启动时间 | 数秒 | 数百 ms~数秒 | 数 ms | 数 ms | 数百 ms |
| 认证 | 少 | 部分 | 多（SafeRTOS） | 增多中（Safety Profile） | 多（汽车/航空/医疗） |

> 注：具体数值高度依赖硬件、配置和测量方法，上表为典型数量级参考。
