# 操作系统 / 网络 / 嵌入式 / 接口 跨域映射

> **目标**：打通 `2.操作系统/`、`3.物联网嵌入式系统/`、`8.网络系统/` 三大领域，建立统一概念映射、机制对照与场景关联。

---

## 1. 概念跨域对照表

| 通用 OS 概念 | Linux 实现 | 嵌入式/RTOS 概念 | 网络系统概念 |
|--------------|------------|------------------|--------------|
| 进程/线程 | `task_struct` / pthread | Task / Thread | - |
| 调度器 | CFS / RT / DL | Fixed Priority / EDF | 流量调度 tc qdisc |
| 地址空间 | `mm_struct` / 页表 | MPU / MMU regions | - |
| 中断 | `request_irq()` / threaded IRQ | ISR / NVIC / GIC | NAPI 软中断 |
| DMA | `dma_map_sg()` / IOMMU | DMA Engine | RDMA / DPDK kernel bypass |
| 设备驱动 | `device_driver` / `platform_driver` | HAL / BSP | NIC driver |
| 设备树 | Device Tree / ACPI | Device Tree / 板级文件 | - |
| 文件系统 | VFS / ext4 / overlayfs | littlefs / FAT | - |
| 网络栈 | `net/` 子系统 | lwIP / FreeRTOS+TCP | TCP/IP / SDN |
| 安全 | LSM / seccomp / capabilities | MPU / TrustZone / secure boot | TLS / IPsec / eBPF |
| 接口 | 系统调用 / ABI / POSIX | CMSIS / HAL API | Socket API / Netlink |

## 2. 数据路径跨域映射

### 2.1 传感器 → 边缘网关 → 云

```mermaid
graph LR
    SENSOR[传感器] -->|I2C/SPI| MCU[MCU / RTOS]
    MCU -->|MQTT/CoAP| EDGE[边缘网关 / Embedded Linux]
    EDGE -->|TCP/IP| CLOUD[云服务器 / Linux]
    CLOUD -->|HTTP/GRPC| APP[应用]
```

| 阶段 | OS/RTOS | 协议 | 接口 |
|------|---------|------|------|
| 传感器采集 | RTOS / Embedded Linux | I2C/SPI/UART | HAL / IIO |
| 边缘处理 | Embedded Linux | MQTT/CoAP | Socket |
| 云端接收 | Linux | TCP/IP | Socket / io_uring |
| 应用服务 | Linux | HTTP/gRPC | socket + userspace proxy |

### 2.2 用户请求 → 容器 → 内核 → 网卡

```mermaid
graph LR
    USER[User Request] -->|HTTP| POD[Container]
    POD -->|veth| BR[Bridge / CNI]
    BR -->|netns| HOST[Host Network Stack]
    HOST -->|TCP/IP| NIC[NIC Driver]
    NIC -->|DMA| PHY[PHY/MAC]
```

## 3. 机制对照

| 机制 | OS 层 | 网络层 | 嵌入式层 |
|------|-------|--------|----------|
| 隔离 | namespace / cgroup | network namespace / VRF | MPU / TrustZone |
| 调度 | CFS / RT | tc qdisc / XDP | 固定优先级调度 |
| 同步 | mutex / futex | conntrack / eBPF map | mutex / semaphore |
| 安全 | LSM / seccomp | TLS / IPsec / eBPF | secure boot / MPU |
| 发现 | udev / sysfs | ARP / ND / BGP | device tree / CANopen |

## 4. 场景关联

| 场景 | 涉及领域 | 关键机制 |
|------|----------|----------|
| 工业物联网 | OS + 嵌入式 + 网络 | RTOS 实时采集 + Linux 网关 + MQTT |
| 车联网 | OS + 嵌入式 + 网络 | CAN + Linux 信息娱乐 + 4G/5G |
| 云原生边缘 | OS + 容器 + 网络 | Kubernetes + containerd + CNI |
| 智能制造 | OS + 嵌入式 + 网络 | PLC + OPC UA + TSN |

## 5. 学习路径贯通

```
计算机组成原理
  → CPU 模式 / 中断 / DMA
    → 操作系统核心（进程/内存/文件/设备）
      → Linux 内核实现
        → 网络协议栈 / 设备驱动 / 安全
          → 嵌入式 Linux / RTOS
            → 物联网协议与边缘计算
              → 分布式系统 / 云原生
```
