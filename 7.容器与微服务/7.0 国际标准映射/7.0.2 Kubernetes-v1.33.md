## 5. 调度与资源语义

项目应在 `7.7 运行时调度语义分析` 中建立 Kubernetes 调度形式化模型：

- **Predicate**：节点筛选（资源、亲和性、污点容忍）。
- **Priority**：节点打分（资源均衡、亲和性）。
- **Preemption**：抢占机制。
- **Scheduling Framework**：v1.33 扩展点（QueueSort、PreFilter、Filter、PostFilter、PreScore、Score、Reserve、Permit、PreBind、Bind、PostBind）。

建议形式化工件：

- TLA+ 规范：Pod 生命周期、Deployment 滚动更新、Scheduler 决策。
- 状态机：Pod Phase（Pending → Running → Succeeded/Failed/Unknown）。
