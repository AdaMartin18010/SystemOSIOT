## 1. 标准概述

| 接口 | 全称 | 版本 | 官方链接 | 适用范围 |
|---|---|---|---|---|
| CNI | Container Network Interface | Spec v1.1.0 | <https://www.cni.dev/docs/spec/> | 容器网络插件 |
| CRI | Container Runtime Interface | Kubernetes API v1 | <https://kubernetes.io/docs/concepts/architecture/cri/> | Kubelet 与运行时 |
| CSI | Container Storage Interface | v1.11+ | <https://github.com/container-storage-interface/spec> | 持久存储插件 |

## 2. CNI Spec v1.1.0

### 2.1 核心接口

| 命令 | 调用时机 | 输入 | 输出 | 对应项目章节 |
|---|---|---|---|---|
| `ADD` | Pod 网络命名空间创建后 | container ID, netns, ifname, config | interfaces, IPs, routes, DNS | 7.1 知识梳理/CNI |
| `DEL` | Pod 销毁前 | container ID, netns, ifname | 无 | 7.1 知识梳理/CNI |
| `CHECK` | CNI v1.1 新增，校验网络配置 | 同 ADD | error / success | 7.1 知识梳理/CNI |
| `VERSION` | 版本协商 | 无 | supported versions | 7.1 知识梳理/CNI |

### 2.2 IPAM（IP Address Management）

| 类型 | 说明 | 对应项目章节 |
|---|---|---|
| host-local | 节点本地 IP 分配 | 7.1 知识梳理/CNI |
| dhcp | DHCP 服务器分配 | 7.1 知识梳理/CNI |
| static | 静态 IP | 7.1 知识梳理/CNI |
| IPv6 / dual-stack | 双栈支持 | 7.1 知识梳理/CNI |

### 2.3 常见 CNI 插件

| 插件 | 模型 | 特性 | 对应项目章节 |
|---|---|---|---|
| Calico | L3 routed (BGP/eBPF) | NetworkPolicy、高性能 | 7.1 知识梳理/CNI |
| Flannel | VXLAN overlay | 简单、无 NetworkPolicy | 7.1 知识梳理/CNI |
| Cilium | eBPF-based | 可观测性、安全、服务网格 | 7.1 知识梳理/CNI |
| Weave | overlay | 加密、易用 | 7.1 知识梳理/CNI |
| Terway | 云原生 ENI | 阿里云实现 | 7.1 知识梳理/CNI |

## 4. CSI（Container Storage Interface）

### 4.1 服务接口

| 服务 | RPC | 职责 | 对应项目章节 |
|---|---|---|---|
| Identity | `GetPluginInfo`, `GetPluginCapabilities`, `Probe` | 插件发现与健康检查 | 7.1 知识梳理/CSI |
| Controller | `CreateVolume`, `DeleteVolume`, `ControllerPublishVolume`, `ControllerUnpublishVolume`, `CreateSnapshot`, `DeleteSnapshot`, `ExpandVolume` | 卷生命周期管理 | 7.1 知识梳理/CSI |
| Node | `NodeStageVolume`, `NodeUnstageVolume`, `NodePublishVolume`, `NodeUnpublishVolume`, `NodeExpandVolume` | 节点挂载/卸载 | 7.1 知识梳理/CSI |

## 5. 接口形式化建议

项目应在 `7.3 形式化结构` 中明确 CNI/CRI/CSI 的接口契约：

- **CNI**：输入/输出 JSON Schema，错误码规范。
- **CRI**：gRPC protobuf 接口定义，状态机（PodSandboxState/ContainerState）。
- **CSI**：控制器/节点/身份服务 RPC 调用序列。

形式化工件建议：

- Alloy 模型：CNI plugin 与 runtime 的接口一致性。
- TLA+：CSI volume attach/detach 状态机。
