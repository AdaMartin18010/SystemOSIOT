# 2.5.3 数学符号

## 1. 操作系统相关数学符号

- OS：操作系统
- K：内核
- P：进程集合
- F：文件系统
- D：设备驱动
- S：系统调用集合
- ⇒：蕴含
- ∀：对所有
- ∃：存在

## 2. 结构化表达

- OS = (K, P, F, D, S)
- K = (资源管理, 调度, 中断处理)
- P = {p1, p2, ..., pn}
- F = (目录, 文件, 权限)
- D = {d1, d2, ..., dm}
- S = {s1, s2, ..., sk}
