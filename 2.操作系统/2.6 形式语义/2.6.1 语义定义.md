# 2.6.1 语义定义

## 1. 语义定义

- 操作系统的语义是指其各组成部分（如进程、内核、系统调用等）在形式化模型下的数学意义和行为约束。
- 形式语义强调可验证性、可推理性和一致性。

## 2. 符号化表达

- OS = (K, P, F, D, S)
- 进程状态：New, Ready, Run, Wait, Term
- 系统调用：S = {s1, s2, ..., sk}

## 3. 结构化表达

- **进程状态机图**：

```mermaid
graph TD;
  New[新建] --> Ready[就绪];
  Ready --> Run[运行];
  Run --> Wait[等待];
  Wait --> Ready;
  Run --> Term[终止];
```
